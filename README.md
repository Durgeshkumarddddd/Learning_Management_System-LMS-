# Learning Management System (LMS)

A **Java-based Learning Management System (LMS)** built using **Object-Oriented Programming (OOP) concepts** and **MVC (Model-View-Controller) architecture**.
This project demonstrates how core Java concepts can be used to design a structured and scalable application.

The system simulates a basic LMS where users can manage learning resources, courses, and interactions between different system components.

---

## 🚀 Features

* Implementation of **MVC Architecture**
* Strong usage of **OOP Concepts**

  * Encapsulation
  * Inheritance
  * Polymorphism
  * Abstraction
* Modular project structure
* Service layer for business logic
* Utility classes for helper operations
* Clean and maintainable code

---

## 🧠 Concepts Used

This project focuses on applying important **Java development concepts**:

* Object-Oriented Programming (OOP)
* MVC Design Pattern
* Layered Architecture
* Separation of Concerns
* Java Collections
* Exception Handling
* Clean Code Principles

## 📂 Project Structure

```
Learning_Management_System-LMS-
│
├── data/           # Stores application data
├── models/         # Entity classes (Course, Student, etc.)
├── services/       # Business logic layer
├── utils/          # Helper and utility classes
│
└── Main.java       # Application entry point
```
